    <h2>1. מה האתר אוסף בעצמו</h2>

    <p>האתר אינו דורש הרשמה, אינו מבקש מידע אישי, ואינו משתמש בעוגיות משלו. כל התכנים פתוחים לקריאה ללא צורך בזיהוי. העדפות המשתמש (מצב כהה/בהיר, מצב קריאה, גודל גופן, הצגת מקור בתצוגה המקבילית) נשמרות ב-<code>localStorage</code> של הדפדפן — זיכרון מקומי שאינו מועבר לשרת.</p>

    <p>פניות שמגיעות לכתובת הקשר נשמרות אך ורק לשם מענה, ואינן מועברות לצד-שלישי כלשהו.</p>

    <h2>2. שירותי צד-שלישי הפועלים באתר</h2>

    <p>האתר משתמש בארבעה שירותי צד-שלישי. כל אחד מהם עשוי להציב עוגיות משלו ולאסוף נתוני גלישה בסיסיים (כתובת IP מקוצרת, סוג דפדפן, מקור התנועה, דפים שנצפו) בהתאם למדיניות הפרטיות שלו.</p>

    <h3>2.1 Google Analytics</h3>
    <p>סטטיסטיקת ביקורים מצטברת — מספר מבקרים, דפים פופולריים, מקורות תנועה. מזהה: <code>G-M3NHG4MWVR</code>. <a href="https://policies.google.com/privacy" rel="noopener">מדיניות הפרטיות של Google</a>.</p>

    <h3>2.2 Microsoft Clarity</h3>
    <p>הקלטות חום (heatmaps) ומסלולי גלילה אנונימיים לשיפור חוויית המשתמש. מזהה פרויקט: <code>ucq1c77qd5</code>. <a href="https://privacy.microsoft.com/en-us/privacystatement" rel="noopener">מדיניות הפרטיות של Microsoft</a>.</p>

    <h3>2.3 Disqus (תגובות)</h3>
    <p>מערכת תגובות חיצונית המוטמעת בתחתית דפי החיבורים. מזהה: <code>hebrew-aramaic-sources</code>. <a href="https://help.disqus.com/en/articles/1717103-disqus-privacy-policy" rel="noopener">מדיניות הפרטיות של Disqus</a>.</p>

    <h3>2.4 Google Translate</h3>
    <p>תרגום אוטומטי. מופעל רק בלחיצה על כפתור "הפעל תרגום אוטומטי" בתחתית האתר — אינו נטען עבור מבקרים שאינם משתמשים בו. <a href="https://policies.google.com/privacy" rel="noopener">מדיניות Google</a>.</p>

    <h2>3. אבטחת מידע</h2>

    <p>האתר מאוחסן על-גבי GitHub Pages המספק חיבור מוצפן (HTTPS). האתר עצמו אינו מכיל טפסים להזנת מידע אישי (למעט מערכת התגובות של Disqus).</p>

    <h2>4. הגבלת טעינת שירותים</h2>

    <p>מי שמעוניין למנוע טעינת שירותי צד-שלישי, יכול להשתמש בתוספי דפדפן כגון <a href="https://github.com/gorhill/uBlock" rel="noopener">uBlock Origin</a>, <a href="https://privacybadger.org/" rel="noopener">Privacy Badger</a>, או מצב "פרטיות מוגברת" מובנה של הדפדפן. כל התכנים באתר נגישים במלואם גם ללא טעינת שירותים אלה.</p>

    <h2>5. ילדים</h2>

    <p>האתר אינו מיועד לילדים ואינו מכוון אליהם, אך תוכנו נגיש לכל גולש ללא הגבלת גיל. איננו אוספים ביודעין מידע אישי מילדים מתחת לגיל 13.</p>
