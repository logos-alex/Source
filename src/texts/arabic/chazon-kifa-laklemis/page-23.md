חָזָק, וְיִהְיֶה בָּאָרֶץ רַעַשׁ גָּדוֹל, וְאַרְאֶה קָדְשִׁי וְאוֹר־
הַמִּזְרָח הָרָץ אֲלֵיהֶם, וְאֶעֱשֶׂה הַמִּזְרָח וְהַמַּעֲרָב מְלֵאֵי אוֹר וְזֹהַר,
וְאַרְאֶה בַּדָּרוֹם עַ״יִן חֲשֵׁכָה נִפְלָאָה. וְאַהֲרֹג בְּאוֹתוֹ הַיּוֹם, אַרְאֶה לַמַּאֲמִינִים
כֹּחַ וְעֹזֶר עַל הַכּוֹפְרִים, כִּי אֲבַעֵר אוֹתָם וְאַעֲשֶׂה פְּנֵיהֶם שְׁחֹרִים
אֲפֵלִים, וְאָסִיר מֵעֲלֵיהֶם אוֹר הַחַיִּים עַד אֲשֶׁר יִהְיוּ הַיְשִׁימוֹן מַסְלִיק מֵהֶם מֵ־
אַחֲרִיתָם, וְאֶשְׁלָחֵם בָּאֲבַדּוֹן, וְהֶעֱבִידוּ הָאֱלִילִים בְּהַרְבּוֹתָם. וַעֲבוֹדָתָם הָאֱלִילִים
בְּלַחְלוּחָם, וְצֶמַח וְהַצְלָחָה הַנְּטוּעָה בַּיָּמִין אֲשֶׁר נְטָעָתָם הַשֵּׂיא,
עַד שֶׁכָּל הַנִּכְנָס לַבַּיִת וְהוּא בּוֹ ― לֹא יִרְצֶה לָשׁוּב אֵלֶיהָ. וְנִכְנְסוּ
בְּנֵי הַבְּרִית בְּאֶרֶץ הַשָּׁ״ם, וְהִגִּיעוּ אֶל הָאָרֶץ הַקְּדוֹשָׁה, וְנֶעֶשְׂתָה
הַתִּקְוָה תַּחַת רַגְלֵיהֶם, וְעָבְרוּ אֶת הַמְּצוּדָה הָעַתִּיקָה אֲשֶׁר גָּבְרוּ עָלֶיהָ בְּנֵי
הַזְּאֵב. וְלֹא יִהְיֶה בֵּינֵיהֶם מִי שֶׁמַּתְפַּלֵּל זוּלָתִי, וְלֹא מִי שֶׁמְּמַחֲצֶה בָּאֲמִירָה, כִּי הִתְכַּ[?]־
נְסוּ הַמַּדְרִיכִים הָאוֹמְרִים עַד שֶׁתְּהֵא אֲמִירָתָם הָאֲמִירָה הָאַחַת, נִתְכַּנְּסוּ עָלֶיהָ
בָּתֵּיהָ הַקְּהִלּוֹת הַקְּדוֹשׁוֹת הַשְּׁלֵמוֹת אֲשֶׁר נִתְכַּנְּסוּ עַל שְׁמִי. וְתֵיעָשֶׂה כְּנֵסִיָּתִי
אַחַת קְהַלִּיָּה שְׁלִיחִית, וּשְׁאָר טִיבוֹתָם, מְטֻהֶרֶת וְנְקִיָּה מִ־
כָּל זוּנִין, וְאִילָן אֶחָד וּפְרִי אֶחָד טוֹב. בַּעֵת
הַהוּא
