### תחילת היחסים --- הרשימה הראשונה

תחילת היחסים: אדם ילד את שת. שת נשא את אקלימא אחות הבל, ותלד לו את אנוש.
אנוש נשא אשה הנקראת חית בת מהומא מזרע חר בן שת, ותלד לו את קינן. קינן
נשא את קרית בת כרשם בן מהיאל, ותלד לו את מהללאל. מהללאל נשא את צחבת פטיר
בת אנוש, ותלד לו את ירד. ירד נשא את זבידא בת כרגלן בן קינן, ותלד לו את
חנוך. חנוך נשא את ירדקין בת טרבח בן מהללאל, ותלד לו את מתושלח. מתושלח
נשא את רחוב בת סרכין בן חנוך, ותלד לו את למך. למך נשא את קיפר בת יוטב בן
מתושלח, ותלד לו את נח. נח נשא את היכל בת משמוס בן חנוך, ותלד לו את שם.
שם נשא את לאה בת נסח, ותלד לו את ארפכשד. ארפכשד נשא את פרדוא בת סלוי בן
יפת, ותלד לו את שלח. שלח נשא את מולדת בת כהן בן שם, ותלד לו את עובד.
עובד נשא את רסדא אחות מלכיצדק בת מלך בן ארפכשד, ותלד לו את פלג. פלג נשא
את חדיב בת חמלח, ותלד לו את יערוע. יערוע נשא את תנעב בת עובד, ותלד לו את
שרוג. שרוג נשא את פיל, ותלד לו את נחור. נחור נשא אשה עקרס בת יערוע, ותלד
לו את תרח. תרח נשא שתי נשים --- האחת יוטא והשנייה סלמת --- ויולד מיוטא
את אברהם ומסלמת את שרה.

אברהם נשא את שרה בת סלמת אשת אביו זו, ותלד לו את יצחק. יצחק נשא את רבקה
בת פטואל, ותלד לו את יעקב. יעקב נשא את לאה בת לבן, ותלד לו את יהודה.
יהודה ילד מתמר את פרץ. פרץ נשא את אפציב בת לוי, ותלד לו את חצרון. חצרון
נשא את פרתיב בת זבולון, ותלד לו את רם. רם נשא את ספוזא בת יהודה, ותלד לו
את עמינדב. עמינדב נשא את ברומא בת חצרון, ותלד לו את נחשון. נחשון נשא את
ארם בת אדם, ותלד לו את שלמון. שלמון נשא את סליב בת עמינדב, ותלד לו את
בועז. בועז נשא את רעות בת לוט, ותלד לו את עובד. עובד נשא את נפות בת
שילא, ותלד לו את ישי. ישי נשא את ארמת בת עתן, ותלד לו את דוד. דוד נשא את
בת שבע בת יוטן בן שילא, ותלד לו את שלמה. שלמה נשא את נעמא בת מחיאל, ותלד
לו את רחבעם.

רחבעם נשא את נחיר בת אל, ותלד לו את אביה. אביה נשא את מעכה בת אבישלום,
ותלד לו את אסא. אסא נשא את עוזבא בת שלח, ותלד לו את יהושפט. יהושפט נשא
את נעמה בת עמון, ותלד לו את יורם. יורם נשא את תלאה בת עמרי, ותלד לו את
אחז. אחז נשא את צומא בת בלהי, ותלד לו את אמוץ. אמוץ נשא את כמא בת קרם,
ותלד לו את עזריה. עזריה נשא את ירוא בת צדוק, ותלד לו את ירים. ירים נשא
את יפת בת הני, ותלד לו את אחז. אחז נשא את אחיר בת זכריה, ותלד לו את
חזקיה. חזקיה נשא את חביבה בת ירמון, ותלד לו את מנשה. מנשה נשא את ארטידא
בת עזריה, ותלד לו את אמון. אמון נשא את טרב בת מורקא, ותלד לו את יאשיהו.
יאשיהו נשא את חמוטל בת ירמיד, ותלד לו את יהואחז. יהואחז נשא אשה, ולא היו
לו ממנה בנים. וימלוך יהויקים אחרי מות אחיו, ונשא אשה הנקראת קרתים בת
חלותא, ותלד לו את שלתיאל. שלתיאל נשא את חמטת בת אליקים, ותלד לו את
זרובבל. זרובבל נשא את מלכות בת עזרא, ותלד לו את ארמיד. ארמיד נשא את
ארקית בת צדוק, ותלד לו את יקים. יקים נשא את חלא בת צורנים, ותלד לו את
עזור. עזור נשא את אפי בת חצור, ותלד לו את צדוק. צדוק נשא את פלטיר בת
דורטיב, ותלד לו את חסדיות. חסדיות נשא את חסגב בת יהוי, ותלד לו את לויד.
לויד נשא את שבשתין בת טובאלי, ותלד לו את אלעזר. אלעזר נשא את הנית בת
יולי, ותלד לו את מתתן. מתתן נשא את סירב בת פנחס, ותלד לו את יעקב.

**ויעקב נשא את חרטיב בת אלעזר, ותלד לו את יהויקים הנקרא יונכר**.[18]

יהויקים נשא את חנה בת פקרדל. ובשישים שנה לנישואיו אותה --- ילדה את מרים
הבתולה אשר ממנה התגלם המשיח. ויוסף הנגר היה בן דודה, ועל כן נפלה גורלו
עליה, כאשר ביקש כהן בני ישראל למסור אותה לאיש אשר יגור בה. וזה היה במסתר
עשה האלוהים גבור ועז, ובעומק דעתו, כי אין ספק כי תחורף מרים הטהורה מן
היהודים בעבור הריונה את המשיח.
