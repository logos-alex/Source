### ספרי הייחוסים והפולמוס

וכאשר אבדו ספרי הייחוסים, נבוכו האבות ביחסיהם, ונתערבבו הדורות אחריהם. ועל כן נאמר לקלימיס כי בירור הייחוסים נעשה מתוך ספרים נסתרים ומסורת רוחנית שנגלתה לקדושים.

המחבר מוסיף כי היהודים החלו לטעון נגדנו שאין אנו מבינים את הייחוסים, ומכוח זה העזו לקלל את אם האור, מרים הבתולה, ולייחס לה זנות. אך רוח הקודש שלימדה את השליחים בעליית ציון היא שלימדה גם את סוד היחסים, כשם שעזרה לעזרא לשנן ולחדש את התורה לאחר אובדן הספרים.

ועל כן ייסתמו פי המלעיזים, כי מרים הטהורה מזרע יהודה, אחר כך מזרע דוד, ואחר כך מזרע אברהם.

### שלוש שריפות הספרים

ונאמר כי ספרי היהודים נשרפו שלוש פעמים:

1. בימי אנטיוכוס, בעת טומאת היכל האדון והקרבת זבחים לעבודה זרה.
2. בימי הורדוס, בשעת חורבן ירושלים.
3. ובפעם נוספת שבה נצרכה עדות רוח הקודש להשבת הזיכרון והייחוס.

ומכאן באה מסירת שרשרת האבות הארוכה: שישים ושלושה אבות הנזכרים בשמותיהם, עד לשבט שממנו התגשם המשיח.

### התחלת הייחוסים: מאדם עד האבות

אדם הוליד את שת; שת נשא את אקלימה, ותלד את אנוש; אנוש את קינן; קינן את מהללאל; מהללאל את ירד; ירד את חנוך; חנוך את מתושלח; מתושלח את למך; למך את נח; נח את שם; שם את ארפכשד; ארפכשד את שלח; שלח את עובד/עבר; עבר את פלג; פלג את יארוע; יארוע את שרוג; שרוג את נחור; נחור את תרח.

תרח נשא שתי נשים: יותה וסלמת. ומיותה נולד אברהם, ומסלמת נולדה שרה. אברהם נשא את שרה והוליד את יצחק; יצחק נשא את רבקה והוליד את יעקב; יעקב נשא את לאה והוליד את יהודה.

### מיהודה עד דוד

יהודה הוליד מתמר את פרץ; פרץ את חצרון; חצרון את ארם; ארם את עמינדב; עמינדב את נחשון; נחשון את שלמון; שלמון את בועז; בועז את עובד; עובד את ישי; ישי את דוד.

דוד נשא את בת־שבע והוליד את שלמה. שלמה הוליד את רחבעם; רחבעם את אביה; אביה את אסא; אסא את יהושפט; יהושפט את יורם; יורם את אחז; אחז את עוזיה; עוזיה את יותם; יותם את אחז; אחז את חזקיה; חזקיה את מנשה; מנשה את אמון; אמון את יאשיהו.

### ממלכי יהודה עד זרובבל

יאשיהו הוליד את יהואחז; אחריו יהויקים; וממנו שאלתיאל; משאלתיאל זרובבל. וזרובבל נשא את מלכות בת עזרא והוליד את ארמיד.

ומכאן ממשיכים הייחוסים לדורות בית שני: ארמיד, ארקית, יקים, עזור, צדוק, יותד, ליוד, אלעזר, מתן — עד הדורות האחרונים המובילים למרים.
