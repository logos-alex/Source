### אובדן ספרי יוחסין ושיקום המסורת

כאשר אבדו ספרי הייחוסים, נתבלבלו האבות והדורות. המחבר אומר לבנו אקלימוס כי עמד על נכונות הייחוסים מתוך ספרים עבריים נסתרים.

עוד הוא טוען שהיו ליהודים טענות נגד מרים, מפני שלדבריהם אין הנוצרים יודעים ייחוסה. כנגד זאת נאמר שהשליחים למדו ברוח הקודש את סודות הייחוס, כשם שעזרא למד ושחזר את ספרי התורה והנביאים.

ועל כן, לפי החיבור, מרים הטהורה היא מזרע יהודה, מזרע דוד ומזרע אברהם; וטענת המלעיזים בטלה.

### שלוש שריפות הספרים לפי המסורת

המחבר מזכיר כי ספרי היהודים נשרפו שלוש פעמים:

1. בימי אנטיוכוס, בטומאת ההיכל;
2. בימי הורדוס וחורבן ירושלים;
3. בזמן נוסף שבו נצרכה מסירה מחודשת על פה וברוח.

ומתוך כך הוא עובר למנות את שושלת היוחסין באריכות כדי לבסס את שרשרת הדורות עד מרים.

### ראשית השושלת עד יהודה ודוד

תחילת הייחוסים: אדם הוליד את שת; שת את אנוש; אנוש את קינן; קינן את מהללאל; מהללאל את ירד; ירד את חנוך; חנוך את מתושלח; מתושלח את למך; למך את נח; נח את שם; שם את ארפכשד; ארפכשד את שלח; שלח את עובד/עבר; עבר את פלג; פלג את רעו; רעו את שרוג; שרוג את נחור; נחור את תרח; תרח את אברהם.

אברהם הוליד את יצחק; יצחק את יעקב; יעקב את יהודה; יהודה (מתמר) את פרץ; פרץ את חצרון; חצרון את ארם; ארם את עמינדב; עמינדב את נחשון; נחשון את שלמון; שלמון את בועז; בועז את עובד; עובד את ישי; ישי את דוד; דוד את שלמה.

### מזרע המלכים עד ימי השיבה

מן שלמה מונה החיבור את שרשרת מלכי יהודה עד גלות בבל: רחבעם, אביה, אסא, יהושפט, יורם, אחז, עוזיהו, יותם, אחז, חזקיהו, מנשה, אמון, יאשיהו, יהואחז, יהויקים ושאלתיאל — עד זרובבל.

ומן שבי בבל: זרובבל נשא את מלכות בת עזרא, והוליד את ארמיד; אחריו נמשכים יקים, עזור, צדוק, יותד, ליוד, אלעזר, מתן ויעקב.
