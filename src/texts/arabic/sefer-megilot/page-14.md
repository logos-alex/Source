### שושלות הייחוס עד מרים

וכאשר אבדו ספרי הייחוסים, התבלבלו האבות בייחוסיהם. ועל כן אומר המחבר לבני קלימיס כי עמד על נכונותם מספרי העברים הנסתרים. והוא מונה את יחס הדורות מאדם ועד מרים הבתולה, כדי לבסס את ייחוסה לשבט יהודה, לבית דוד ולזרע אברהם.

ראשית הייחוסים: אדם הוליד את שת; שת את אנוש; אנוש את קינן; קינן את מהללאל; מהללאל את ירד; ירד את חנוך; חנוך את מתושלח; מתושלח את למך; למך את נח; נח את שם; שם את ארפכשד; ארפכשד את שלח; שלח את עבר; עבר את פלג; פלג את רעו; רעו את שרוג; שרוג את נחור; נחור את תרח; תרח את אברהם; אברהם את יצחק; יצחק את יעקב; יעקב את יהודה; יהודה את פרץ; פרץ את חצרון; חצרון את ארם; ארם את עמינדב; עמינדב את נחשון; נחשון את שלמון; שלמון את בועז; בועז את עובד; עובד את ישי; ישי את דוד; דוד את שלמה; ושלשלת המלכים עד ימי הגלות והשיבה — עד זרובבל.

ומן השיבה ממשיך החיבור יוחסין מפורט לדורות בית שני: זרובבל, אביהוד, יהויקים, צדוק, אטין, טור, אלעזר, מתן, יעקב, יוסף, ויהויקים הידוע ביונאכר; ולבסוף חנה יולדת את מרים הבתולה, אשר ממנה התגשם אדוננו המשיח.

ועל כן החיבור חותם בפולמוס נגד הטוענים כי מרים אינה מזרע יהודה: רוח הקודש היא שלימדה את השליחים את סוד הייחוסים, כשם שלימדה את עזריה את חוק האדון לאחר אבדן הספרים. ולפיכך — לפי מחבר החיבור — נסתמת טענת המלעיזים, ומרים הטהורה עומדת בתוך ייחוס יהודה, דוד ואברהם.

ולאדוננו ואלוהינו ישוע המשיח השבח והכוח והגדולה וההדר וההשתחוויה עם האב ורוח הקודש, מעתה ובכל עת ולעולם ועד. אמן.
