### המשך מסוף פרק יג

וכשנאבדו ספרי היחסים נבהלו האבות ביחס ונבהלו אחריהם בזה, עד שעמדתי על נכונותם מספרי העברים הנסתרים. ואני אספר לך בני אקלימוס זאת: כאשר בא זרובבל אל ירושלים, נשא את מלכה בת עזרא המורה ותלד לו בן ויקרא שמו אביהוד, והייתה זו לפניו אשת יהויכין.

וכשגדל אביהוד נשא את רגליב בת יהושע בן יוצדק הכהן, ותלד לו בן ויקרא שמו יוֹאָקִים. ויישא יוֹאָקִים אישה ותלד לו בן, וכשגדל נשא את אלגית בת חצרון ותלד לו את צדוק. ויישא צדוק את פלבין בת רחב ותלד לו את אתין. ויישא אתין את חסכיב בת יולע ותלד לו את תור. ויישא תור את סלסין בת חסול ותלד לו את אלעזר. ויישא אלעזר את הבית בת מלח ותלד לו את מנר.

### ראשית היחוסים הארוכים

ולמען ידיעתנו, בני אקלימוס, ביחס הגבירה מרים וביחס אבותיה, מתחילים היהודים בטענות עלינו שאין אנו מבינים את היחסים ולא יודעים אותם, ומעיזים לקלל את אם האור הגבירה מרים הבתולה ומייחסים אותה לזנות.

כי אינם יודעים שרוח הקודש אשר ירדה עלינו, חברת שנים־עשר בעלייית ציון, היא שלימדה אותנו את כל מה שהיינו צריכים לדעת מן היחסים ושאר הסתרים, כמו שלימדה את עזריה מורה התורה כולה עד ששנה אותה וחידשה.

ועתה ייסתמו פי היהודים הארורים, ויידעו כי מרים הטהורה מזרע יהודה ואחר כך מזרע דוד ואחר כך מזרע אברהם. ואין להם טענה על היחסים שלימדה אותנו רוח הקודש, ולא נשאר בידם ספר שיוכלו לדעת ממנו יחס, כי ספריהם נשרפו שלוש פעמים: הראשונה בימי אנטיוכוס אשר טימא את היכל ה' וציווה להקריב זבחים לעבודה זרה; והשנייה בימי הורדוס בעת חורבן ירושלים; והשלישית – ושמע אתה הבן המבורך מה שהשרתה עלי רוח הקודש – שישים ושלושה אבות הרשומים בשמותיהם.

### התחלת היחסים

אדם הוליד את שת. שת נשא את אקלימה אחות הבל ותלד לו את אנוש. אנוש נשא אישה ששמה חית בת מהוממא מזרע חר בן שת ותלד לו את קינן. קינן נשא את קרית בת כרשם בן מהיאל ותלד לו את מהללאל. מהללאל נשא את צחב בת אנוש ותלד לו את ירד. ירד נשא את זבידה בת כרגלן בן קינן ותלד לו את חנוך. חנוך נשא את ירדקין בת טרבח בן מהללאל ותלד לו את מתושלח. מתושלח נשא את רחוב בת סרכין בן חנוך ותלד לו את למך. למך נשא את קיפר בת יותב בן מתושלח ותלד לו את נח.

נח נשא את היכל בת משאמוס בן חנוך ותלד לו את שם. שם נשא את לייה בת נצח ותלד לו את ארפכשד. ארפכשד נשא את פרדוא בת סלואה בן יפת ותלד לו את שלח. שלח נשא את מולדת בת כהן בן שם ותלד לו את עובד. עובד נשא את רסדה אחות מלכיצדק בת מלכה בן ארפכשד ותלד לו את פלג. פלג נשא את חדיב בת חמלח ותלד לו את יארוע. יארוע נשא את תנעב בת עובד ותלד לו את שרוג. שרוג נשא את פיל ותלד לו את נחור. נחור נשא אישה עקרס בת ארוע ותלד לו את תרח.

תרח נשא שתי נשים: יותה וסלמת. ויולד מיותה את אברהם ומסלמת את שרה. אברהם נשא את שרה בת סלמת אשת אביו זו ותלד לו את יצחק. יצחק נשא את רבקה בת פתואל ותלד לו את יעקב. יעקב נשא את לאה בת לבן ותלד לו את יהודה. יהודה הוליד מתמר את פרץ. פרץ בן יהודה נשא את אפציב בת לוי ותלד לו את חצרון. חצרון נשא את פרטיב בת זבולון ותלד לו את ארם. ארם נשא את ספוזה בת יהודה ותלד לו את עמינדב. עמינדב נשא את ברומה בת חצרון ותלד לו את נחשון. נחשון נשא את ארם בת אדם ותלד לו את שלמון. שלמון נשא את סליב בת עמינדב ותלד לו את בועז. בועז נשא את ארעוף בת לוט ותלד לו את עובד. עובד נשא את נפות בת שילה ותלד לו את ישי. ישי נשא אישה בת עתן ותלד לו את דוד.

דוד נשא את בת־שבע בת יותן בן שילה ותלד לו את שלמה. שלמה נשא את נעמה בת מחיל ותלד לו את רחבעם. רחבעם נשא את נחיר בת אל ותלד לו את אביה. אביה נשא את מעכה בת אבישלום ותלד לו את אסא. אסא נשא את עוזבה בת שלח ותלד לו את יהושפט. יהושפט נשא את נעמנה בת עמון ותלד לו את יורם. יורם נשא את תלעיה בת עמוי ותלד לו את אחז. אחז נשא את צומה בת בלהי ותלד לו את עמוס. עמוס נשא את קמה בת קראם ותלד לו את עוזיה. עוזיה נשא את ירואה בת צדוק ותלד לו את יריים. יריים נשא את יעפת בת הני ותלד לו את אחז. אחז נשא את אחיר בת זכריה ותלד לו את חזקיה. חזקיה נשא את חביבה בת ירמון ותלד לו את מנשה. מנשה נשא את ארטידה בת עזריה ותלד לו את אמון. אמון נשא את תרב בת מורקא ותלד לו את יאשיהו.
