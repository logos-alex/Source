נח נשא את היכל בת משאמוס בן חנוך ותלד לו את שם. שם נשא את לייה בת נצח ותלד לו את ארפכשד. ארפכשד נשא את פרדוא בת סלואה בן יפת ותלד לו את שלח. שלח נשא את מולדת בת כהן בן שם ותלד לו את עובד. עובד נשא את רסדה אחות מלכיצדק בת מלכה בן ארפכשד ותלד לו את פלג. פלג נשא את חדיב בת חמלח ותלד לו את יארוע. יארוע נשא את תנעב בת עובד ותלד לו את שרוג. שרוג נשא את פיל ותלד לו את נחור. נחור נשא אישה עקרס בת ארוע ותלד לו את תרח.

תרח נשא שתי נשים: יותה וסלמת. ויולד מיותה את אברהם ומסלמת את שרה. אברהם נשא את שרה בת סלמת אשת אביו זו ותלד לו את יצחק. יצחק נשא אישה ששמה רבקה בת פתואל ותלד לו את יעקב. יעקב נשא את לאה בת לבן ותלד לו את יהודה. יהודה הוליד מתמר את פרץ. ויישא פרץ בן יהודה את אפציב בת לוי ותלד לו את חצרון. חצרון נשא את פרטיב בת זבולון ותלד לו את ארם. ארם נשא את ספוזה בת יהודה ותלד לו את עמינדב. עמינדב נשא את ברומה בת חצרון ותלד לו את נחשון. נחשון נשא את ארם בת אדם ותלד לו את שלמון. שלמון נשא את סליב בת עמינדב ותלד לו את בועז. בועז נשא את ארעוף בת לוט ותלד לו את עובד. עובד נשא את נפות בת שילה ותלד לו את ישי. ישי נשא אישה בת עתן ותלד לו את דוד.

דוד נשא את בת שבע בת יותן בן שילה ותלד לו את שלמה. שלמה נשא את נעמה בת מחיל ותלד לו את רחבעם. אשר לא היה לו ממנה אלא רחבעם. רחבעם נשא את נחיר בת אל ותלד לו את אביה. אביה נשא את מעכה בת אבישלום ותלד לו את אסא. אסא נשא את עוזבה בת שלח ותלד לו את יהושפט. יהושפט נשא את נעמנה בת עמון ותלד לו את יורם. יורם נשא את תלעיה בת עמוי ותלד לו את אחז. אחז נשא את צומה בת בלהי ותלד לו את עמוס. עמוס נשא את קמה בת קראם ותלד לו את עוזיה. עוזיה נשא את ירואה בת צדוק ותלד לו את יריים. יריים נשא את יעפת בת הני ותלד לו את אחז. אחז נשא את אחיר בת זכריה ותלד לו את חזקיה. חזקיה נשא את חביבה בת ירמון ותלד לו את מנשה. מנשה נשא את ארטידה בת עזריה ותלד לו את אמון. אמון נשא את תרב בת מורקא ותלד לו את יאשיהו.

יאשיהו נשא את חמטול בת ארמיד ותלד לו את יואחז. יואחז נשא אישה ולא היו לו ממנה בנים, ומלך יוֹאָקִים אחרי מות אחיו. ויישא יוֹאָקִים אישה ששמה קרטים בת חלותא ותלד לו את שאלתיאל. ושאלתיאל נשא את חמטת בת אליקים ותלד לו את זרובבל. זרובבל נשא את מלכות בת עזרא ותלד לו את ארמיד. וארמיד נשא את ארקית בת צדוק ותלד לו את יוֹאָקִים. ויוֹאָקִים נשא את חלי בת זורנים ותלד לו את עזור. עזור נשא את אבי בת חצור ותלד לו את צדוק. צדוק נשא את פלטיר בת דורטיב ותלד לו את אסהם־יותד. יותד־אסהם נשא את חסגב בת יוֹאי ותלד לו את ליוד. ליוד נשא את שבשתין בת חובה בלייה ותלד לו את אלעזר. אלעזר נשא את הנית בת יולי ותלד לו את מתן.
