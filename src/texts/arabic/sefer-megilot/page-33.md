### הגנאלוגיה המפורטת — 63 האבות (חלק א)

[ibtida al-ansab] אדם הוליד את שת. שת נשא לאשה את אקלימה אחות הבל, וילדה לו את אנוש.

אנוש נשא לאשה אישה ששמה חיט בת מהמומא מבני חר בן שת, וילדה לו את קינן.
קינן נשא לאשה את קרית בת כרשם בן מהיאל, וילדה לו את מהללאל.
מהללאל נשא לאשה את צחב פטר בת אנוש, וילדה לו את ירד.
ירד נשא לאשה את זבידה בת כרגלן בן קינן, וילדה לו את חנוך.
וחנוך נשא לאשה את ירדקין בת טרבח בן מהללאל, וילדה לו את מתושלח.
מתושלח נשא לאשה את רחוב בת סרכין בן חנוך, וילדה לו את למך.
למך נשא לאשה את קיפר בת יוטב בן מתושלח, וילדה לו את נח.
נח נשא לאשה את היכל בת מושאמוס בן חנוך, וילדה לו את שם.
שם נשא לאשה את ליאה בת נצח, וילדה לו את ארפכשד.
ארפכשד נשא לאשה את פרדואה בת סלואה בן יפת, וילדה לו את שלח.
שלח נשא לאשה את מולדת (בת) כהן בן שם, וילדה לו את עובד.
עובד נשא לאשה את רסדה אחות מלכיצדק בת מלך בן ארפכשד, וילדה לו את פלג.
פלג נשא לאשה את חדיב בת חמלאח, וילדה לו את יערו.
יערו נשא לאשה את תנעאב בת עובד, וילדה לו את שרוג.
שרוג נשא לאשה את פיל, וילדה לו את נחור.
נחור נשא לאשה את אישה עקרה בת ארוע, וילדה לו את תרח.
תרח נשא שתי נשים, אחת שמה יוטה והשנייה שלמה, ומיוטה נולד לו אברהם, ומשלמה נולדה לו שרה.
אברהם נשא את שרה בת שלמה אשת אביו, וילדה לו את יצחק.
יצחק נשא לאשה אישה ששמה רבקה בת פתואל, וילדה לו את יעקב.
יעקב נשא את לאה בת לבן, וילדה לו את יהודה.
יהודה הוליד את פרץ מתמר. ופרץ בן יהודה נשא את אפדיב בת לוי, וילדה לו את חצרון.
חצרון נשא את פרטיב בת אזבלון, וילדה לו את ארם.
ארם נשא את ספוזה בת יהודה, וילדה לו את עמינדב.
עמינדב נשא את ברומא בת חצרון, וילדה לו את נחשון.
נחשון נשא את ארם בת אדם, וילדה לו את שלמה.
שלמה נשא את שליב בת עמינדב, וילדה לו את בעז.
בעז נשא את ארעוף בת לוט, וילדה לו את עובד.
עובד נשא את נפות בת שילה, וילדה לו את אישי.
אישי נשא את אמראת בת עתן, וילדה לו את דוד.
דוד נשא את בת-שבע בת יוטן בן שילה, וילדה לו את שלמה.
שלמה נשא את נעמה בת מאחל, וילדה לו את רחבעם (ולא היה לה בן אחר מלבד רחבעם).
רחבעם נשא את נחיר בת אל, וילדה לו את אביה.
אביה נשא את מעכה בת אבשלום, וילדה לו את אסא.
אסא נשא את עוזבא בת שלח, וילדה לו את יהושפט.
יהושפט נשא את נעמנה בת אמון, וילדה לו את יורם.
יורם נשא את תלעיה בת עמרי, וילדה לו את אחז.
אחז נשא את צומא בת בלהי, וילדה לו את אמוץ.
אמוץ נשא את קאמא בת קרם, וילדה לו את עוזיה.
