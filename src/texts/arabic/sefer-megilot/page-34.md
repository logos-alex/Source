### הגנאלוגיה המפורטת — 63 האבות (חלק ב) וחתימה

עוזיה נשא את ירואה בת צדוק, וילדה לו את יריום.
יריום נשא את יעפת בת חני, וילדה לו את אחז.
אחז נשא את אחיר בת זכריה, וילדה לו את חזקיה.
חזקיה נשא את חביבה בת ירמון, וילדה לו את מנשה.
מנשה נשא את ארתידא בת עזוריה, וילדה לו את אמון.
אמון נשא את תארב בת מורקא, וילדה לו את יאשיהו.
יאשיהו נשא את חמטול בת ארמיד, וילדה לו את יואחז.
יואחז נשא אישה ולא היו לו ממנה בנים. ומלך יאקים אחרי אחיו, ונשא אישה ששמה קרטים בת חלוטא, וילדה לו את שלאתיאל.
ושלאתיאל נשא את חמטת בת היאקים, וילדה לו את זרובבל.
זרובבל נשא את מלכות בת עזרא, וילדה לו את ארמיד.
וארמיד נשא את ארקית בת צדוק, וילדה לו את יאקים.
ויאקים נשא את חלי בת צורנים, וילדה לו את עזור.
עזור נשא את אפי בת חצור, וילדה לו את צדוק.
צדוק נשא את פלטיר בת דורטיב, וילדה לו את אשם יוטיד.
אשם יוטיד נשא את חסגאב בת יואי, וילדה לו את ליוד.
ליוד נשא את שבשתין בת חובא בליא, וילדה לו את אלעזר.
אלעזר נשא את חנית בת יולי, וילדה לו את מתן.
מתן נשא את סיראב בת פנחס, וילדה לו את יעקב.
ויעקב נשא את חרטיב בת אלעזר, וילדה לו את יואקים הידוע ביונכר.
יואקים נשא את חנה. נחזור לבית אלעזר, ואחרי ששים שנה מנשואיו אליה, ילדה לו את מרים הבתולה שממנה התגשם המשיח.

הערה: יוסף הנגר היה בן דודה מצד האב, ולכן נפלה גורלו עליה כאשר רצה כהן בני ישראל למסור אותה לאיש שיכול לכסות אותה. והיה בסתר מעשה האלוהים, יתעלה וישתבח, ונסתרה ידיעתו, שמרים הטהורה צריכה להיות מושמצת על ידי היהודים בגלל הריונה במשיח.

לכבודו ואלוהינו ואדונינו ישוע המשיח, התשבחה והגבורה והגדולה וההדר והסגידה, עם האב ורוח הקודש, מעתה ובכל עת ועד סוף הדורות. אמן.
